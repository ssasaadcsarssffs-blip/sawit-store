Apa itu Panel Pterodactyl?

Panel Pterodactyl adalah panel manajemen server open-source yang digunakan untuk menjalankan game server atau bot (seperti bot Discord/WA) di dalam Linux container yang terisolasi dengan aman dan mudah.

Bagaimana cara pemasangan Script Bot WA?

Kamu hanya perlu melakukan instalasi Node.js, melakukan clone atau upload script ke panel/vps, lalu jalankan perintah `npm start` dan scan QR code menggunakan WhatsApp kamu.
